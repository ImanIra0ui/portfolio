Projects

Obstacle Avoidance Based-Visual Navigation

Python (OpenCV), ROS, Webots

A simple visual navigation approach to obstacle avoidance and goal search. Given a robot equipped with a camera which it uses to capture images of its surroundings, the goal is to process the images to detect the obstacles and the target without relying on any other inputs. The performance of the robot is simulated using Webots to demonstrate that it is able to successfully accomplish this task in different setups.

Neb's Bakery Web Application

Python (Django)

Web application designed to automate the management of a bakery in Kenitra, Morocco. Some of its functionalities include product listing, placing orders, and transaction management for admin users.

Robust Intention Aware Mapless Crowd Navigation Using 2D Range Sensors

Python

Crowd navigation state-of-the-art method, AEMCARL , is combined with realistic LiDAR sensor readings that offer position estimates of the humans in the scene instead of the ground truth, since the expectation of perfect information about the positions is unrealistic. The goal of this project is to address this key limitation while replicating robust crowd-navigation performance.

Alpha Language Center Web and Desktop Applications

Java (Java Servlet API, Java Swing), PostgreSQL

Web and desktop applications to automate the management of a language center, acting like a learning management system. This software manages transactions, organises the teachers' and students' schedules, and has a submission space for students. It also automates the manager's tasks including enrolling/dropping students to/from modules, and keeping track of the teachers' work hours.

Adversarial Bias Mitigation

Python (Keras)

The adversarial debiasing approach, proposed by Zhang et al. , is first replicated on the income prediction task with gender and marital status as protected variables. It is also applied to the the recidivism prediction task, using race as the protected attribute, to investigate the technique’s effectiveness with non-binary protected variables. The aim is to determine whether adversarial debiasing could generalise to different tasks and protected variables.

Leveraging NLP for Domestic Abuse Analysis

Python (scikit-learn)

The performances of different classifiers (SVM, Naive Bayes, Logistic Regression) are compared in the detection of domestic abuse in Reddit posts. Frame semantics are then used on posts containing abusive situations to extract information from the posts, and provide insight into the dynamics of domestic abuse (parties involved, causes, etc.). The results are compared to clinical findings in the field of social sciences to confirm their accuracy.
